This project presents an interactive Mobile Sales Analytics Dashboard developed using Microsoft Power BI to evaluate sales performance across mobile brands, models, regions, and time periods.

The dataset is cleaned, transformed, and modeled using Power Query, while DAX formulas are used to build key KPIs such as total sales, profit, quantity sold, and performance trends. The dashboard enables data-driven decision-making through intuitive visuals and dynamic filtering.
